Portal Ramadhan: Pusat Ibadah Digital

Portal Ramadhan adalah aplikasi berbasis web yang dirancang untuk membantu produktivitas ibadah harian selama bulan suci. Proyek ini dikembangkan dengan arsitektur murni untuk memastikan performa yang ringan dan responsif.

## 1. Deskripsi Proyek
Aplikasi ini berfungsi sebagai hub navigasi terpadu untuk mengakses berbagai alat bantu ibadah digital. Fokus utama proyek adalah kemudahan akses dan pemantauan kemajuan ibadah secara *real-time*.

## 2. Teknologi yang Digunakan
Proyek ini dibangun menggunakan *stack* teknologi *frontend* murni:
* **HTML5**: Struktur semantik dokumen.
* **CSS3**: Desain antarmuka responsif dengan variabel kustom.
* **Vanilla JavaScript**: Logika fungsional dan manipulasi DOM interaktif.

## 3. Fitur Utama
Sistem telah mengimplementasikan fitur-fitur berikut:
* **Kumpulan Doa**: Referensi bacaan doa dan niat harian.
* **Tasbih Digital**: Penghitung dzikir dengan fitur target yang dapat disesuaikan.
* **Kalkulator Zakat**: Perhitungan otomatis berdasarkan nisab emas terkini.
* **To-Do List Ibadah**: Pemantauan progres harian terintegrasi.
* **Jadwal Imsakiyah**: Jadwal imsak, berbuka, dan shalat dari berbagai daerah.


## 4. Instruksi Instalasi Lokal
Untuk menjalankan proyek di lingkungan lokal:
1. Pastikan Anda memiliki **VS Code** dan ekstensi **Live Server**.
2. *Clone* repositori ini atau unduh file HTML terkait.
3. Klik kanan pada file `index.html` dan pilih **"Open with Live Server"**.
4. Aplikasi akan berjalan pada alamat `http://127.0.0.1:5500`.

## 5. Kompatibilitas
Aplikasi telah melalui tahap pengujian dan terverifikasi kompatibel dengan:
* **Google Chrome** (Desktop & Mobile)
* **Safari** (iOS & macOS)
